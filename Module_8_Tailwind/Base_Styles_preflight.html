This is a simple page using Tailwind CSS with Preflight enabled.

Introduction

Tailwind's preflight base styles help normalize CSS across different browsers and provide a consistent starting point for your designs.

How Preflight Helps

Preflight applies a set of default styles to your HTML elements, like setting `box-sizing` to `border-box` for all elements, removing list bullets, resetting link styles, and other common browser resets.

  • Consistency in typography
  • Consistent box model
  • Resets for form elements
  • Accessible buttons
